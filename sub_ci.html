회사 소개 페이지

클리오 코스메틱스는 더 나은
품질과 서비스를 제공하는 글로벌
No.1 기업으로 성장하겠습니다

CI

LOGO

Word Mark

클리오 코스메틱스는 ‘끌리다’라는 단어에서 영감을 얻어 제작된 화장품 전문 제조기업입니다.
화장품을 통하여 마음을 이끌어 내고 대중에게 눈길을 끌게 하다는 의미에서 클리오라고 정의하게 되었습니다.
클리오는 세상을 바꾸는 아름다움을 창조하기를 원합니다.

* 브랜드 이미지를 훼손시키지 않는 범위에 따라 컬러 변경 및 스케일 조정이 가능합니다.

SYMBOL

Symbol Mark

클리오 코스메틱스의 심벌은 기업의 가치의 핵심인 변화가 반영된 ‘나비’입니다.
외적으로는 주목성과 화려함을, 외적인 아름다움을 내적으로는 새로운 변화, 비상하다의 뜻을 담고 있습니다.
심벌에 사용된 메인 컬러인 오렌지는 기업의 정체성과 정통성을 상징하며,
자연스러운 그라데이션을 사용하여 부드럽게 조화를 이루고 있습니다.

* 브랜드 이미지를 훼손시키지 않는 범위에 따라 컬러 변경 및 스케일 조정이 가능합니다.

COLOR

기업의 정체성인 새로움, 자신감, 즐거움을 표현하는 오렌지 컬러를 메인으로 사용합니다.
문화의 활력과 화려함을 상징하는 서브 핑크 컬러와 같이 조화롭게 나타낼 수 있습니다.
짙은 그레이 컬러는 혁신을 상징하며 신뢰감을 나타냅니다.

  • 정체성

    1. RGB 255 155 111
    2. CMYK 0 50 56 0
    3. WEB #ff976f

  • 문화

    1. RGB 255 94 112
    2. CMYK 0 78 42 0
    3. WEB #ff5e70

  • 신뢰성

    1. RGB 43 43 43
    2. CMYK 70 64 63 65
    3. WEB #2b2b2b
